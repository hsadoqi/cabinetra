# @cabinetra/ui-hooks

Shared React hooks for the Cabinetra UI system.

## Installation

```bash
pnpm add @cabinetra/ui-hooks
```

## Usage

```tsx
import { useI18n, useTheme, useAuth } from "@cabinetra/ui-hooks";
```

## Hooks

- `useI18n` - Internationalization
- `useTheme` - Theme management
- `useAuth` - Authentication
